# Implementation Plan - v8.4.3: Local & WebGUI Analyzer Updates

This plan consolidates the updates for both the local React analyzer (`ResumeAnalyzer-local.jsx`) and the Claude Artifact WebGUI (`ResumeAnalyze-webgui.jsx`) to align with `v8.4.2` terminology and `v8.3.1` functional requirements.

## Goal Description
Update both analyzer interfaces to reflect the new "Tool-First" terminology (removing "Phase" numbering), enforce portfolio labeling guardrails, and improve version visibility.

## User Review Required
> [!IMPORTANT]
> **Portfolio Labeling Logic**: The system will use a simplified heuristic (checking for "Project" in title or Position 0) to append `(Independent Project)` until a dedicated flag is available in the parser.
> **Deployment**: WebGUI updates in `claude-artifacts/` are intended for copy-pasting back into Claude chats or standalone tools.
