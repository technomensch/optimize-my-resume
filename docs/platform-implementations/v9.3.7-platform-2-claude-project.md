Do NOT run bullet generation in a single turn.

### Turn 1: Planning
**Prompt:** "Reference `constrained-generation-workflow.md` Turn 1. Review my job history and JD. Generate the Unified Planning Table and Keyword Distribution Plan. STOP before generating any bullets."

### Turn 2: Execution
**Prompt:** "Approved. Execute Turn 2. Generate bullets following the `guardrail-validation-schema.md` JSON format. Conclude with the Final Reconciliation Table."
